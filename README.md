## 🚦 ICT 紀律規則

### 進場後 4 條鐵律
1. 進場後不動 SL
2. 觸 TP 立刻出 (不貪)
3. 觸 SL 立刻認 (不拗)
4. NY Lunch (00-01) 不操作

### 6 個不開單情境
- 時段不對 (NY Lunch / Off-session)
- Daily 跟方向反
- 沒 Judas 訊號
- 流動性沒被掃
- 心情 < 5 / 連虧 3 筆當日
- RR < 目標
